## 📄 Resume Download

To add your resume:
1. Save your resume as `resume.pdf`.
2. Upload it to this repo (same directory as `index.html`).
3. Add a download link in your HTML file like this:
   ```html
   <a href="resume.pdf" download>Download My Resume</a>
   ```
